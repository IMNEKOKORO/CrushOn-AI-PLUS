## üìñ User Guide

 **AUTOMATIC SCENES**  
Music changes automatically based on the scene card.  
The name you give to the scene card must be placed in the **Keyword** field.

 **SOUND EFFECTS (SFX)**  
Configure short sounds for actions.  
They will play when the **Keyword** is mentioned by **{{user}}** or **{{char}}**.

 **VISUALS AND FILTERS**  
Assign animated emojis (Rain, Vortex, etc.) and screen effects (Flash, Shake, Blood, etc.) that activate when the **Keyword** appears in the conversation.

üîó **AUDIO LINKS GUIDE**  
‚Ä¢ **YouTube:** Paste only the end of the link starting from `v=`  
  Example: `v=DZJpqqkSce0`  
‚Ä¢ **Dropbox:** Replace `www.dropbox.com` with `dl.dropboxusercontent.com` to enable direct audio playback.

 **DICE AND COIN**  
Useful for RPG roles or quick decisions.

 **NOTEBOOK**  
You can use the notebook to write important things like "keywords" that the character needs or things you don't want to forget, practically whatever you want.

 **THEMES**  
several custom themes so you can use the one you like the most (this option is a beta, if the project has a lot of support, in the future it will have more customization such as modifying the colors of the page)
 
 **BACKUP**  
Use **Export** to download your `.json` configuration and **Import** to restore it or share it with others.
